# docs/decisions

These records are this kit's own architectural decision history (ADRs). Each
file documents a design choice made during kit development, the rationale, and
the alternatives considered.

They are not examples to copy into a target repository. When adopting this kit,
your target repository gets `000-template.md` as the starting shape. Create your
own `docs/decisions/*.md` records there as your project makes architectural
decisions worth preserving.

See `docs/checklists/decision-failure-memory.md` for guidance on when a change
warrants a new decision record versus a failure record or a final-report note.
